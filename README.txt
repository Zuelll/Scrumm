# Scrumm

A video game with the following goals
-Allow the practice of Scrum (including short introductions and complementary documentation)
-Set this practice in a “realistic” simulation” (the degree of realism depends on the difficulty/available effort)
-Include a certain level of entertainment (gamification) with a gradual advance in the difficulty.


## Getting Started

Game still in development, accessible with Unity.
